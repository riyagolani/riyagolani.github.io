Projects

Image Tag Gallery

A full-stack application that allows you to upload an image and uses Google Cloud Vision API to generate tags corresponding to the image. Those tags serve as metadata for the image that can be searched for. The backend consists of REST APIs to upload, process and search for images developed using Express and images are stored in AWS S3 for accessibility. The frontend, developed using React, allows users to upload an image and view the image along with the metadata in the image gallery and search for specific images by tags.
Tech Stack: React, Express, Google Cloud Vision API, AWS S3

Netflix Clone

Created a Netflix clone project, meticulously crafted using Tailwind CSS and Django.
