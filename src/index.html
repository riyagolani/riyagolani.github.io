Netflix Clone

Created a Netflix clone project, meticulously crafted using Tailwind CSS and Django. This endeavor involved replicating the original platform's aesthetics and functionality, achieving an impressive 60% similarity in both user interface and features compared to the authentic Netflix experience. Leveraging Django's robust authentication system, I fortified the application against CSRF attacks, ensuring the security of user interactions. Furthermore, I utilized SQLite3 as the database engine to store essential data such as user profiles, movies, and videos, facilitating seamless retrieval and management of information.
Tech Stack: Django, Tailwind CSS, Python
